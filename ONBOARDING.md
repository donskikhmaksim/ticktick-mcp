# Подключить TickTick MCP к своему Claude

Тебе прислали персональную команду для терминала — она настроит всё автоматически.
Займёт около 10 минут.

---

## Что тебе нужно заранее

- **Аккаунт Railway** — создай на [railway.app](https://railway.app) (бесплатно, можно через GitHub)
- **Аккаунт GitHub** — нужен для Railway (если нет — [зарегистрируйся](https://github.com), 2 минуты)
- **Chrome** — для получения одного токена вручную (шаг 4 в скрипте)
- **Claude Pro / Team** — нужна возможность добавлять коннекторы

---

## Установка

Возьми команду, которую тебе прислали, и запусти её в терминале:

**Mac:** открой Spotlight (Cmd+Space) → набери `Terminal` → Enter  
**Windows:** нажми Win+R → набери `cmd` → Enter

Вставь команду и нажми Enter. Скрипт сам:

1. Установит Railway CLI
2. Попросит войти в Railway (откроется браузер)
3. Откроет страницу входа TickTick — войди и скопируй токены
4. Покажет как достать куку из Chrome (2 минуты)
5. Создаст проект на Railway и задеплоит сервер
6. Выдаст готовую ссылку для Claude

---

## Последний шаг — добавить коннектор в Claude

После того как скрипт завершится, он выдаст ссылку вида:
```
https://ticktick-mcp-xxxx.up.railway.app/mcp/xxxxxxxx
```

1. Открой [claude.ai](https://claude.ai) → профиль → **Settings** → **Connectors**
2. Нажми **Add custom connector**
3. Вставь ссылку → **Save**

Готово. Напиши Claude: _«Покажи мои проекты в TickTick»_ — и убедись что всё работает.

---

## Если что-то пошло не так

**Скрипт упал с ошибкой** — запусти его заново, он безопасен для повторного запуска.

**Claude говорит «не могу подключиться»** — проверь что ссылка в коннекторе скопирована целиком (включая `/mcp/...` в конце).

**Railway просит оплату** — выбери Hobby план (бесплатный), он подходит.

**Не вижу вкладку Application в Chrome** — DevTools могут быть узкими, попробуй растянуть панель.

Если не помогло — напиши тому, кто прислал тебе команду.
