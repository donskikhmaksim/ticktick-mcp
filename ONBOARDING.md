# Подключить TickTick MCP к своему Claude

Тебе прислали персональную команду для терминала. Вставляешь её один раз —
скрипт всё настроит сам. Займёт 5–7 минут, и почти всё время — просто ожидание.

---

## Что нужно заранее

- **Аккаунт Railway** — если нет, создай на [railway.app](https://railway.app) (бесплатно, можно через GitHub)
- **Аккаунт GitHub** — нужен для входа в Railway (если нет — [зарегистрируйся](https://github.com), 2 минуты)
- **Chrome** — только если захочешь расширенные функции (шаг 4, необязательный)
- **Claude Pro / Team** — чтобы можно было добавлять коннекторы

---

## Как запустить

### 1. Открой терминал

**Mac:** нажми `Cmd + Space`, набери `Terminal`, нажми Enter
**Windows:** нажми `Win + R`, набери `cmd`, нажми Enter

### 2. Вставь команду, которую тебе прислали

Она выглядит примерно так (у тебя будет своя, с реальными ключами):

```bash
bash <(curl -fsSL https://github.com/donskikhmaksim/ticktick-mcp/raw/main/scripts/setup.sh) \
  --client-id "..." \
  --client-secret "..." \
  --timezone "Europe/Moscow"
```

Нажми Enter. Дальше скрипт всё делает сам и подсказывает, что нажимать.

---

## Что произойдёт по шагам

**Шаг 1 — Railway CLI.** Скрипт сам установит нужную программу, если её нет. Ничего делать не нужно.

**Шаг 2 — Вход в Railway.** Откроется браузер — войди в свой аккаунт Railway (или зарегистрируйся, если ещё не был). Вернись в терминал и жди.

**Шаг 3 — Деплой.** Скрипт создаёт твой персональный сервер и загружает его на Railway. Занимает 1–2 минуты, просто подожди.

**Шаг 4 — Вход в TickTick.** Откроется браузер со страницей входа TickTick — **важно: войди именно в свой аккаунт**. Нажми Allow. На экране появится надпись «TickTick подключён» — это значит всё сработало. Вернись в терминал и нажми Enter.

**Шаг 5 — Расширенные функции (необязательно).** Скрипт спросит, хочешь ли включить теги, привычки, корзину и перемещение задач между списками. Если да — попросит куку из Chrome (единственный ручной шаг во всей установке):

1. Открой ticktick.com в Chrome, залогинься
2. Нажми `F12` (на Mac: `Option + Cmd + I`)
3. Вкладка **Application** → слева **Storage → Cookies → https://ticktick.com**
4. В поле Filter введи `t`
5. Найди строку с именем ровно **`t`** (одна буква)
6. Двойной клик по значению → скопируй
7. Вставь в терминал

Если не хочешь сейчас — просто нажми `n`, можно будет включить позже (спроси у того, кто дал инструкцию, как).

---

## В конце

Скрипт покажет ссылку вида:
```
https://ticktick-mcp-production-xxxx.up.railway.app/mcp/xxxxxxxx
```

Возьми её и:

1. Открой [claude.ai](https://claude.ai) → иконка профиля → **Settings** → **Connectors**
2. **Add custom connector**
3. Вставь ссылку → **Save**

Проверь: напиши Claude **«Покажи мои проекты в TickTick»**. Если видишь список — готово.

---

## Если что-то не так

**Скрипт упал с ошибкой** — просто запусти команду ещё раз, это безопасно.

**Браузер открылся не туда / показал ошибку** — убедись, что не заблокировал всплывающие окна для терминала/браузера, и попробуй снова.

**Claude не подключается** — проверь, что ссылка скопирована целиком, включая `/mcp/...` в конце.

**Не видно вкладку Application в Chrome** — расширь окно DevTools.

Если не помогло — напиши тому, кто прислал команду.
